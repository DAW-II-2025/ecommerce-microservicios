Política de Privacidad

Esta aplicación es desarrollada y gestionada por Multiservicios El Triangulo, y está comprometida con la protección de los datos personales de los usuarios.

1. Información que recopilamos

Podemos recopilar información como nombre, número de teléfono, mensajes enviados y otra información proporcionada voluntariamente por el usuario durante el uso de nuestra aplicación.

2. Uso de la información

Utilizamos la información recopilada exclusivamente para brindar los servicios ofrecidos por la aplicación, como la atención automática mediante bots o gestión de pedidos. No compartimos esta información con terceros sin consentimiento explícito.

3. Retención y eliminación

Los datos se conservarán durante el tiempo necesario para cumplir con los fines para los que fueron recopilados. El usuario puede solicitar la eliminación de sus datos en cualquier momento contactándonos.

4. Seguridad

Implementamos medidas técnicas y organizativas para proteger los datos personales contra el acceso no autorizado, pérdida o destrucción.

5. Derechos del usuario

El usuario tiene derecho a acceder, rectificar, eliminar o limitar el uso de sus datos personales. Para ejercer estos derechos, puede comunicarse con nosotros.
